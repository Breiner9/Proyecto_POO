# Proyecto POO - Python

## 📌 Descripción
Proyecto desarrollado en Python para aplicar los conceptos fundamentales de Programación Orientada a Objetos (POO) mediante una serie de ejercicios prácticos que incluyen clases, herencia, encapsulamiento, polimorfismo, abstracción, interfaces, composición y manejo de excepciones.

---

- Python

## 🚀 Cómo ejecutar
1. Abrir la terminal  
2. Ubicarse en la carpeta del proyecto  
3. Ejecutar:

python main.py

## 📚 Conceptos aplicados (según la guía)
- Clases, objetos y métodos  
- Encapsulamiento y modificadores de acceso  
- Constructores e inicialización de objetos  
- Herencia (clases padre e hijo)  
- Sobreescritura de métodos  
- Polimorfismo  
- Clases abstractas y métodos abstractos  
- Interfaces (implementadas con clases abstractas)  
- Composición (relación entre clases)  
- Manejo de excepciones  
- Excepciones personalizadas  

---

## 🧪 Ejercicios implementados
- Ejercicio 1: Creación de una clase básica (Coche)  
- Ejercicio 2: Encapsulamiento (getters y setters)  
- Ejercicio 3: Constructores  
- Ejercicio 4: Herencia (Vehículo, Coche, Bicicleta)  
- Ejercicio 5: Sobreescritura de métodos  
- Ejercicio 6: Polimorfismo con listas de objetos  
- Ejercicio 7: Clases abstractas (Animal, Perro, Gato)  
- Ejercicio 8: Interfaces (Volador, Pájaro, Avión)  
- Ejercicio 9: Composición (Motor dentro de Coche)  
- Ejercicio 10: Manejo de excepciones personalizadas
